"Remove Max Width" is a plugin designed to remove the Max-width from the WordPress 2.5 Admin interface.

== Description ==

With WordPress 2.5, A new Admin interface has come around. And while it looks great, Theres one thing i disliked about it.
The Maximum width of the page being limited to 980px, This plugin removes that design choice, and allows for maximum screen realestate use.

For those users who wish to have a smaller tinyMce editing window, While still retaining the ability to allow things to spread out over the page width, this plugin also enables you to resize TinyMCE's editing windows Horizontally as well as the default standard of Virtical.

Note: Version 1.3 is an optional update.
