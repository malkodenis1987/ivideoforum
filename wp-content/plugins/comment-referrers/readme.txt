== Description ==
Adds the referrering link from which a commenting user came from to comment notifications.

Note: if you're using Jetpack Comments the referrer information is lost and unfortunately this plugin won't work correctly.

== Installation ==
Copy into your plugins folder and activate. If you are using a caching plugin such as [WP Super Cache](http://ocaoimh.ie/wp-super-cache/) make sure you clear the cache after enabling this plugin.
